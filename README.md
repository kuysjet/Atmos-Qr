# 📱 Atmos-QR

A web-based QR Code Attendance Management System designed to make attendance faster, more accurate, and paperless. Instead of manually signing attendance sheets or waiting in long lines, users simply display or scan a QR code to record their attendance instantly.

---

## 🚀 Purpose

Atmos-QR was developed to simplify and automate attendance monitoring.

Traditional attendance methods often require participants to:
- Wait in long lines
- Sign attendance sheets manually
- Spend unnecessary time checking in

With Atmos-QR, attendance only takes a few seconds.

Users simply present their unique QR Code, and the system records their attendance instantly after scanning.

---

## ✨ Features

- QR Code-based attendance
- Fast and paperless check-in
- Student and employee registration
- Attendance history
- Real-time attendance records
- Event attendance monitoring
- Admin dashboard
- Attendance reports
- QR Code generation
- Secure login system

---

## ⚙️ How It Works

1. Register users in the system.
2. Generate a unique QR Code for each user.
3. Open the attendance scanner.
4. Users present their QR Code.
5. The QR Code is scanned.
6. Attendance is automatically recorded in the database.
7. Administrators can view attendance records and generate reports.
